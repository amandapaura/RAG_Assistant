🔍 Docker:
1. Parar tudo
docker-compose down -v
                ↓
2. Limpar completamente o cache e imagens corrompidas
docker system prune -a --volumes
                ↓
Vai perguntar se tem certeza, digite: y
                ↓
3. Verificar se limpou
docker images
docker ps -a
                ↓
4. Rebuild do zero
docker-compose build --no-cache
                ↓
5. Subir os serviços
docker-compose up
